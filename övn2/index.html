Vi kan skapa utfyllnadstext i VS code med att skriva lorem och sedan trycka RETUR

Detta ger 20 utfyllnadsord

Vi kan bestäma hur många utfyllnadsord genom att skriva LoremNR exempelvis lorem6 ger sex ord

Om vi vill ha väldigt många ord så kan vi skriva lorem*NR vilket multiplicera med 20

Exempelvis om vi skriver lorem*3 får vi 20*3
